**知识点：** git的使用方法

**概念：** 

1. 设置姓名和email
2. git init	文件初始化
3. touch .gitignore 设置忽略文件
4. git status 状态命令
5. git add 增加文件
6. git commit -m""  提交增加的文件
7. git reset --hard HEAD~n 回退n个版本；git reset --hard  commitID 回退到某个版本
8. git reflog 查看每一次记录   git reset --hard commitID  可以回退到任意版本
